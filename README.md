
# 贾斯纳的猫
贾斯纳的猫，一个学习类别和逻辑的网络游戏


我和3岁的孩子一起做这个游戏是为了发展逻辑语言技能。它使用布尔逻辑从数到越来越难的单词问题。在她目前的发展水平上，这似乎是她学习的有效途径，我在其他地方找不到这样的东西。

Enjoy!
